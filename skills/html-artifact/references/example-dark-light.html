Typography & code

Body prose uses var(--fg) against var(--bg). Inline code spans use the dedicated --code-bg / --code-fg pair. Block code:

:root {
  color-scheme: light dark;
  --bg: #ffffff;
  --fg: #1f1f1f;
}
@media (prefers-color-scheme: dark) {
  :root { --bg: #1e1e1e; --fg: #cccccc; }
}
